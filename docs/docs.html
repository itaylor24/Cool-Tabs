Cool Tabs Documentation

Overview

Cool Tabs is a web component that makes tabs easier. Cool Tabs has three modes: Plain, Block, and Underline. Tabs can also be oriented to the left or right.

HTML

Cool Tabs consists of three custom elements: cool-tab, cool-tab-item, and cool-tab-content. cool-tab is the main custom element, and cool-tab-item and cool-tab-content are its child elements.

  • Cool-Tab

    • Properties

      • Type

        A cool tab can have three types: "plain", "block", or "underline" which change the type of tab item displayed. The default type is "plain." Any other value given to the type attribute other than the listed three will change to plain. Although the tab-items have classes that mirror cool-tab type, they will always be the same as the parent cool-tab.

      • Side

        A cool tab can have can be on one of two sides: "left" or "right". This property changes the orientation of the tabs.

      • Select

        The select property determines which tab's content to display. It takes a number which corresponds to the value of the tab item to display content for. Only one tab should be selected at a time

  • Cool-Tab-Item

    • Properties

      • Live

        Live is a boolean attribute that determines whether a tab is selected. If a tab is live, its associated content is displayed.

      • Frozen

        Frozen is a boolean attribute that determines whether a tab can be interacted with. Frozen tabs do not recieve pointer events.

      • Value

        A tab's value links a tab to its cool-tab-content. Each tab should have a unique value. Each tab-content pair should have the same value.

      Contents

      A tab-items conents should be the content to display in the tab label. This can be anything, though different content will effect the width and height of the tab.

  • Cool-Tab-Content

    • Properties

      • Value

        A tab content's value links it to its tab-item equivalent. Each tab should have a unique value. Each tab-content pair should have the same value.

      Contents

      A tab-content's content should be the content to display when selecting the corresponding tab. This content can be anything.

CSS

Cool Tabs have lots of custom properties because of the three types. To add large amounts of customization, it is recommended to use the plain tab type. Cool Tabs styles should be added via css file named 'cool-tabs-css.css'.

                    class CoolTab extends HTMLElement{
                        constructor(){
                            super(); 
                            const shadow = this.attachShadow({mode: 'open'}); 
                    
                            this.shadowRoot.innerHTML = `
                                
                                
    
                                    
                                
    
                                
                                
                            `   
                        }
                    
                        #select; 
                        #type; 
                        #orientation =this.getAttribute('orientation'); 
                        #side = this.getAttribute('side'); 
                        
                        get select() {
                            return this.getAttribute('select');
                        }
                        
                        set select(newValue) {
                            this.setAttribute('select', newValue);
                        }
                    
                        get orientation() {
                            return this.getAttribute('orientation');
                        }
                        
                        set orientation(newValue) {
                            this.setAttribute('orientation', newValue);
                        }
                    
                        get side() {
                            return this.getAttribute('side');
                        }
                        
                        set side(newValue) {
                            this.setAttribute('side', newValue);
                        }
                    
                        get type() {
                            return this.getAttribute('type');
                        }
                        
                        set type(newValue) {
                            this.setAttribute('type', newValue);
                        }
                    
                    
                        static get observedAttributes() { return ["select","type","side","orientation"] }
                    
                        attributeChangedCallback(name, oldValue, newValue){
                            if(name === "select"){
                                this.#select = newValue; 
                            }
                            if(name === "type"){
                                this.#type = newValue; 
                            }
                            if(name === "side"){
                                this.#side = newValue; 
                            }
                            if(name === "orientation"){
                                this.#orientation = newValue; 
                            }
                            this.#renderTabs();  
                        }    
                    
                        connectedCallback(){
                            if(!this.hasAttribute("type") || !["plain","underline","block"].includes(this.getAttribute("type"))){
                                this.setAttribute("type","plain"); 
                            }
                            this.#renderTabs(); 
                        }
                    
                        #renderTabs(){
                            if(!this.hasAttribute("type") || !["plain","underline","block"].includes(this.getAttribute("type"))){
                                this.setAttribute("type","plain"); 
                            }
                            this.shadowRoot.firstElementChild.nextElementSibling.setAttribute("class","cooltabs");
                            
                            let thisChildren = Array.from(this.children);  
                    
                            thisChildren.forEach(child=>{ 
                                child.setAttribute("type",this.#type); 
                                child.removeAttribute("selected"); 
                            })
                    
                            let childToSelect = thisChildren.filter(child=>{
                                return child.getAttribute('value') === this.#select &&!child.hasAttribute("frozen")
                            });  
                    
                            if (childToSelect[0]!= undefined){
                                childToSelect[0].setAttribute("selected",''); 
                            }
                            
                            this.shadowRoot.firstElementChild.nextElementSibling.classList.add(this.#orientation); 
                            this.shadowRoot.firstElementChild.nextElementSibling.classList.add(this.#side);
                    
                        }
                    }
                    
                    class CoolTabItem extends HTMLElement{
                        constructor(){
                            
                            super(); 
                            this.setAttribute("slot","tabs"); 
                            const shadow = this.attachShadow({mode: 'open'}); 
                            this.shadowRoot.innerHTML =`
                                
                                
    
                                    
    
                                
    
                            `
                            
                        }
                        #value = this.getAttribute('value'); 
                        #selected = this.getAttribute("live"); 
                        #type = this.getAttribute('type'); 
                        #frozen = this.getAttribute('frozen'); 
                    
                        get value() {
                            return this.getAttribute('value');
                        }
                        
                        set value(newValue) {
                            this.setAttribute('value', newValue);
                        }
                    
                        get live(){
                            return this.getAttribute("live"); 
                        }
                        set live(newValue){
                            this.setAttribute("live",newValue); 
                            
                        }
                    
                        get frozen() {
                            return this.getAttribute('frozen');
                        }
                        set frozen(newValue) {
                            this.setAttribute('frozen', newValue);
                        }
                    
                        static get observedAttributes() { return ["selected","type",'frozen'] }
                    
                        attributeChangedCallback(name, oldValue, newValue){
                            if(name === "selected"){
                                this.#selected = newValue; 
                            }
                            if(name === "type"){
                                this.#type = newValue; 
                            }
                            if(name==="frozen"){
                                this.#frozen = newValue; 
                                if(newValue){
                                    this.addEventListener("click",this.#selectTab); 
                                }
                            }
                    
                            this.#renderTab(); 
                        }  
                        
                        connectedCallback(){
                     
                    
                            if(this.hasAttribute('live')){
                                this.parentElement.setAttribute("select",this.value); 
                                this.#selected = true; 
                            }
                            if(!this.hasAttribute('frozen')){
                                this.addEventListener("click",this.#selectTab)
                            }
                           
                            this.#renderTab(); 
                            
                        }
                    
                        #renderTab(){
                            this.shadowRoot.lastElementChild.setAttribute("class","tabwrapper"); 
                    
                            if(this.hasAttribute('selected')){
                                this.shadowRoot.lastElementChild.classList.add("selected"); 
                               
                            }else{
                                this.shadowRoot.lastElementChild.classList.remove("selected"); 
                            }
                    
                            if (this.hasAttribute('frozen')){
                                this.shadowRoot.lastElementChild.classList.add("frozen"); 
                            }else{
                                this.shadowRoot.lastElementChild.classList.remove("frozen");
                            }
                    
                            if(this.hasAttribute("type")){
                                this.shadowRoot.lastElementChild.classList.add(this.#type); 
                            }
                            
                        }
                        #selectTab(evt){
                            let coolTabSelection = new CustomEvent('coolTabSelection'); 
                            this.dispatchEvent(coolTabSelection);
                            this.parentElement.setAttribute("select",this.value); 
                        }
                    }
                    
                    class CoolTabContent extends HTMLElement{
                        constructor(){
                            super(); 
                            const shadow = this.attachShadow({mode: 'open'}); 
                            this.shadowRoot.innerHTML=`
                            
                            
    
                            `
                        }
                        #value; 
                        get value() {
                            return this.getAttribute('value');
                        }
                        set value(newValue) {
                            this.setAttribute('value', newValue);
                        }
                        static get observedAttributes() { return ['value'] }
                    
                        attributeChangedCallback(name, oldValue, newValue){
                            if(name === "value"){
                                this.#value = newValue; 
                            }
                            this.#renderContent(); 
                        }  
                    
                        connectedCallback(){
                            this.#renderContent();
                            let me = this; 
                            const observerParent = new MutationObserver(mutations=>{
                                mutations.forEach(function(mutation) {
                                    if (mutation.type === "attributes") {
                                        me.#renderContent(); 
                                    }
                                }); 
                            });
                         
                            observerParent.observe(this.parentElement, {
                                attributes: true //configure it to listen to attribute changes
                            });
                            
                        }
                        #renderContent(){
                            this.shadowRoot.lastElementChild.setAttribute("class","");
                            this.shadowRoot.lastElementChild.classList.add(this.parentElement.getAttribute("side")+'side');
                          
                            if(this.parentElement.getAttribute("select")!=this.#value){
                                this.shadowRoot.lastElementChild.classList.add("hidden"); 
                            }else{
                                this.shadowRoot.lastElementChild.classList.remove("hidden"); 
                            }
                        }
                    }
